# Evidence Certification

For every material claim, locate its source and reproduction path. Classify it as `verified`, `partially_verified`, `unverified`, or `contradicted`. Check freshness, environment, sample limits, access constraints, and whether the evidence actually supports the stated conclusion. A missing source is not neutral evidence. Produce a certificate containing claim coverage, failed checks, residual uncertainty, and an explicit release recommendation for the Lead score gate.
